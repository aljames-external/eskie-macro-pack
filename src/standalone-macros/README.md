The macros contained in this directory are "standalone" in the sense that they can be copied and pasted into the Foundry VTT macro editor and run without needing to install the `eskie-macros` module (EMP). 

They were created at a fixed moment in time and are not updated regularly (if at all). 
When (not if) they break you'll have to request it be fixed or fix it yourself!

These are generated from the module source and are loosely tested. They are meant to largely be a way for you to modify macros to your liking if the module does not support it. If you create something amazing please post it on the discord and it might become officially supported in the module!
